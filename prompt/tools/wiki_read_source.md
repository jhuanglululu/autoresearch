Read a source's full text and metadata from the immutable evidence layer. Read-only. A
retracted source carries a retraction banner, and the output lists which summaries cite it.

Sources are the only evidence: fetched URLs (re-checkable) and experiment runs
(reproducible) outweigh plain captures (another agent's testimony).

Id format: lowercase with hyphens, no spaces (e.g. "arxiv-1706-03762"). Use
`wiki_search_sources` first when you don't know the exact id.
